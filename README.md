# Yarukoto
ローカルで動くシンプルなタスク管理アプリケーション

Yarukoto（ヤルコト） は、あなたの頭の中にあるタスクをすべて書き出してスッキリさせるためのタスク管理デスクトップアプリです。
ローカルにデータを保存するため、内容を気にせず安心して入力いただけます。
インストール不要で利用いただけます。

主な機能と特徴
シンプルな操作性: 直感的なUIで、タスクの追加・確認・完了がサクッと行えます。
親子タスクの階層化: 大きなタスクはドラッグ＆ドロップで小さなタスクに細分化できます。
強力なルーティン（日課）機能: 毎日・毎週・毎月の繰り返しタスクを自動生成。さらに内閣府の最新祝日データと連動し、「祝日はスキップ」「祝日は前倒し」といった高度なスケジュール設定が可能です。
ダッシュボード分析: 全体のタスク完了率や、直近7日間の達成傾向をグラフで視覚化し、モチベーションを維持します（β版機能です）。
完全なプライバシー保護: データは外部サーバー（クラウド）には一切送信されず、すべてご利用のPC内（ローカル環境のSQLite）に安全に保存されます。
